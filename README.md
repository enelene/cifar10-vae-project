# CIFAR-10 Generative Models Project
This repository contains the implementation of a Variational Autoencoder (VAE) from scratch.

## Structure
- `src/model.py`: Manual implementation of VAE (Encoder/Decoder).
- `src/dataset.py`: Data loading and preprocessing.
- `src/utils.py`: Loss functions (ELBO) and visualization tools.
- `src/train.py`: Training engine.
